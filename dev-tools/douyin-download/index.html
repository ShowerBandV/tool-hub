🎵 抖音视频下载

粘贴抖音分享链接 · 在线解析无水印视频 · 支持多种画质 · 纯前端解析

📖 使用说明

  1. 在抖音 App 中点击分享按钮，复制视频链接。
  2. 将链接粘贴到上方输入框中（支持短链 v.douyin.com 和长链 douyin.com/video/）。
  3. 点击「解析视频」按钮，工具通过 API 获取视频信息和下载地址。
  4. 在结果列表中选择需要的画质，点击「下载」按钮保存视频到本地。
⚠️ 本工具仅用于解析公开视频的下载链接，请遵守抖音平台的使用条款和相关法律法规。请勿用于下载付费内容或侵犯他人版权。
