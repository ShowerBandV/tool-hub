📖 SQL格式化工具介绍

SQL格式化工具用于美化杂乱无章的SQL语句，使其更易阅读和维护。工具自动识别SQL中的关键字（SELECT、FROM、WHERE、JOIN、GROUP BY、ORDER BY等），将每个关键字换行并对齐缩进，子句层级关系一目了然。支持常见SQL方言（MySQL、PostgreSQL、SQLite等）。"压缩SQL"功能可将格式化的SQL还原为单行紧凑格式，方便在代码中嵌入或API参数中传输。格式化完全在浏览器本地完成，SQL语句不会上传至任何服务器。

📝 使用步骤

  1. 粘贴SQL — 将需要格式化的SQL语句粘贴到上方文本框中。
  2. 格式化 — 点击"🔧 格式化"按钮，语句自动换行缩进排版。
  3. 压缩（可选） — 点击"压缩SQL"将美观格式还原为单行紧凑格式。
  4. 复制结果 — 点击"📋 复制结果"将格式化后的SQL保存到剪贴板。

💡 适用场景

  • 代码审查 — 格式化同事或开源项目中的复杂SQL进行审查。
  • SQL教学 — 展示SQL查询的结构和关键字层级关系。
