🔐 Base64 编解码

在线Base64编解码工具。支持文本与Base64字符串双向互转，也可将图片文件拖入自动转为Base64编码。上传图片即可获得可嵌入HTML的Data URL，适合前端开发场景。


📖 Base64 编解码介绍

Base64 是一种将二进制数据转换为 ASCII 字符串的编码方式，广泛应用于网络数据传输、邮件附件（MIME）、HTML/CSS 中嵌入图片（data URI）等场景。本工具支持文本的 Base64 编码和解码，以及图片文件与 Base64 字符串之间的双向转换。编码过程是可逆的，解码后还原为原始数据。所有处理在浏览器本地完成，敏感数据不会上传至任何服务器。

📝 使用步骤

  1. 选择模式 — 切换「文本编码」「文本解码」「图片转Base64」或「Base64转图片」模式。
  2. 输入数据 — 在文本框中输入待处理的字符串或选择图片文件。
  3. 复制结果 — 编码/解码结果实时显示，点击复制按钮即可使用。

💡 适用场景

  • 前端开发 — 将小图标转为 Base64 嵌入 CSS，减少 HTTP 请求。
  • 数据传输 — 在 JSON 或 URL 中安全传输二进制数据。
