🔗 URL 编解码

在线URL编解码工具。输入文本一键进行encodeURIComponent编码或decodeURIComponent解码，自动识别百分号编码字符。适合处理URL参数、API请求中的特殊字符转义。

📖 URL编解码工具说明

URL编码（Percent-encoding）将URL中的特殊字符转换为百分号加两位十六进制数的格式，确保URL在传输中的安全性。本工具支持 encodeURI / encodeURIComponent / decodeURI / decodeURIComponent 四种方式实时转换。encodeURIComponent 会对更多字符编码（包括 /?&=+# 等保留字符），适合编码查询参数值。输入即转换，结果实时显示，适合前端开发处理URL参数和API请求中的特殊字符。

📝 使用步骤

  1. 输入文本 — 粘贴要编码或解码的字符串。
  2. 选择模式 — 切换编码或解码模式。
  3. 复制结果 — 结果实时显示，点击复制按钮即可使用。

💡 适用场景

  • API调试 — 编码查询参数确保特殊字符正确传输。
  • 网页开发 — 处理用户输入中的特殊字符防止URL注入。
