文件盒子

File Toolbox · 在线文件处理工具 — 纯前端处理，文件不上传服务器，保护隐私安全

文件盒子提供了 2 种常用在线文件处理工具：ZIP 压缩打包和解压提取、文件 MD5 哈希值校验。ZIP 工具支持选择多个文件打包为 ZIP 压缩包下载，也支持上传 ZIP 文件解压提取内部文件。MD5 工具通过拖拽文件计算文件的 MD5 哈希值，用于验证下载文件的完整性。所有操作在浏览器本地完成，文件不上传服务器，保护文件隐私安全。
