📖 关于工具盒子

Tool Hub · 一站式在线工具集，致力于提供免费、好用、无需安装的在线工具。

🎯 站点用途

工具盒子是一个轻量级在线工具集合网站，汇集了计算器、开发者工具、小游戏与趣味测试等多个类别的实用工具。所有工具均为纯前端实现，无需注册登录，即开即用。

📦 工具分类

  • 计算器盒子 — 学业计算（GPA、成绩转换、AP估分）、科学/图形计算器、单位换算（长度、重量、温度、面积、数据）、生肖查询、人民币大小写、拼音字母表、家庭称谓计算等。
  • 开发者盒子 — JSON格式化、Base64编解码、正则测试、密码/UUID生成、时间戳、哈希计算、URL编解码、二维码、跑马灯、随机点名、文本批量替换等。
  • 游戏盒子 — 涂鸦跳跃、反应速度测试、色盲测试等休闲游戏与趣味测验。

🔒 隐私声明

本网站高度重视用户隐私：

  • 所有工具均在浏览器本地运行，用户输入的数据不会上传到任何服务器。
  • 不使用 Cookie 追踪用户行为（除必要的广告展示 Cookie）。
  • 游戏分数、计算器历史等数据仅存储在浏览器本地存储（localStorage）中，用户可随时清除。
  • 本网站使用 Google AdSense 展示广告，Google 可能使用 Cookie 来投放个性化广告。详情请参阅 Google 广告政策。
  • 本网站不收集、不存储任何个人身份信息。

🛠️ 技术架构

纯静态 HTML/CSS/JavaScript 构建，托管于 GitHub Pages。无后端服务器，无数据库，所有功能完全在浏览器中实现。使用原生 DOM API 和少量第三方库（如 anime.js 用于动画效果）。
