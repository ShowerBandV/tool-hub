📖 帧打包工具功能介绍

帧打包工具是一个在线视频帧处理工具，支持从视频中按时间间隔提取帧画面、对绿幕背景进行颜色替换或透明处理、基于感知哈希算法（dHash）智能去除相似帧，以及将处理后的帧序列导出为 PNG 图片包（ZIP）或 GIF 动画。所有处理均在浏览器本地完成，视频数据不上传服务器，保护您的隐私安全。工具适用于游戏素材制作、动画帧编辑、视频内容分析等场景。

📝 使用步骤

  1. 选择视频 — 点击"选择视频文件"按钮，从本地选取一个视频文件（支持 MP4、WebM 等常见格式）。
  2. 设置参数 — 调整"抽帧间隔"（秒）控制抽帧密度，调整"去重敏感度"控制相似帧判定标准。
  3. 开始抽帧 — 点击"🎬 开始抽帧"按钮，工具会按设定间隔逐帧提取画面并显示缩略图。
  4. 背景编辑（可选） — 选择背景色、替换色和容差，点击"应用于所有帧"进行绿幕抠像。
  5. 智能去重（可选） — 点击"🔍 智能去重"按钮自动移除高度相似的帧。
  6. 导出 — 点击"📦 导出序列帧 (ZIP)"下载 PNG 序列帧，或点击"🎞️ 导出 GIF"生成动画 GIF。

💡 适用场景

  • 游戏素材 — 从视频中提取角色动画帧，制作 2D 游戏精灵图或 Sprite Sheet。
  • 动画制作 — 将视频片段转为 GIF 动画，用于产品展示、表情包或社交媒体内容。
  • 视频分析 — 按时间间隔抽取关键帧，快速浏览视频内容或建立视频摘要。
  • 课件制作 — 从教学视频中提取重点画面，制作图文教程或演示文稿素材。
⚠️ 处理大视频时建议降低抽帧密度（增大抽帧间隔）。GIF 生成较慢，帧数建议控制在 100 帧以内。导出透明背景的 GIF 时请确保已勾选"导出透明背景"。
